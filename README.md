## 1. Introduction & Setup
Basic environment testing and your first script.
* **Topic:** `print()` function and basic execution.
* **Example:** `first.py` [cite: 1]
```python
print("Hello World")
```

## 2. Data Types & Variables
Understanding how Python stores information.
* **Integers & Floats:** `x = 5`, `y = 3.14`
* **Strings:** `name = "Aniket"` [cite: 1]
* **Booleans:** `is_learning = True`

## 3. Control Flow
Making decisions in your code.
* **If-Else Statements:** Conditional logic.
* **Loops:** `for` loops for iteration and `while` loops for condition-based execution.

## 4. Functions & Modules
Writing reusable code and importing external libraries.
* **Defining Functions:** `def my_function():`
* **External Modules:** Using `pip` to install packages like `pyjokes` [cite: 1].
* **Example:** `module.py` [cite: 1]
```python
import pyjokes
print(pyjokes.get_joke())
```

## 5. Data Structures
Organizing data efficiently.
* **Lists:** Ordered, mutable collections.
* **Tuples:** Ordered, immutable collections.
* **Dictionaries:** Key-value pairs.
* **Sets:** Unique elements.

## 6. Object-Oriented Programming (OOP)
Building complex systems using classes and objects.
* **Classes & Objects:** Blueprints for data.
* **Inheritance:** Reusing code from parent classes.
* **Encapsulation & Polymorphism.**

## 7. Error Handling
Preventing your program from crashing.
* **Try-Except Blocks:** Handling exceptions gracefully.
```python
try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")
```

## 8. File I/O
Reading from and writing to files.
* **Open/Read/Write:** Handling `.txt`, `.csv`, and `.json` files.

## 9. Advanced Topics
* **List Comprehensions:** Concise list creation.
* **Decorators:** Modifying function behavior.
* **Generators:** Memory-efficient iteration.
